# ClassVault — Student Management Dashboard

ClassVault is a modern, responsive **student management dashboard** built using vanilla HTML, CSS, and JavaScript. It provides a clean UI for adding, editing, filtering, and tracking student records with persistent local storage and real-time UI updates.

---

## 🚀 Features

### Core Functionality
- Add new students with validation (Name, Class, Timing)
- Edit existing student records via modal interface
- Delete students with instant UI update
- Persistent data storage using `localStorage`

### Advanced UI Features
- Real-time search by student name
- Class-based filtering
- Sorting options:
  - Recent additions
  - Alphabetical (Name A–Z)
  - Class-based sorting

### UI/UX Enhancements
- Responsive grid layout (mobile + desktop)
- Dark / Light mode toggle
- Animated card rendering and transitions
